Cómo usar esta página

⚠️ Aviso:
Si estás usando esta página desde un celular, te recomendamos activar la opción “Sitio para computadoras” (o "Ver como escritorio") en tu navegador para una mejor visualización.

  1. Crear materias Obligatorio antes de usar

    Abrí 🎨 Materias, añadí el nombre y elegí un color. Podés editarlas o borrarlas cuando quieras.

  2. Registrar estudio (📒 Estudio)

    Tocá un día del calendario para agregar horas/minutos, tema y materia. Se pueden cargar varias materias por día. Las etiquetas muestran un tooltip con tiempo y tema.

  3. Planificar la semana (🗓️ Agenda)

    En la vista Agenda añadís clases con horario por día (08:00–23:00). Los bloques se pintan con el color de la materia.

  4. Gestionar parciales (📝 Parciales)

    Calendario separado para exámenes. Hacé clic en la fecha para cargar materia, tema y observaciones. No interfiere con el calendario de estudio.

  5. Gestión de materias

    Recordá que las materias pueden editarse y eliminarse desde 🎨 Materias. Si eliminás una materia, los registros existentes permanecerán pero la materia ya no aparecerá en la lista para seleccionar.
